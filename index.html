Principais TAGS do HTML e suas funcionalidades:

  • !DOCTYPE html "..." : Essa Tag é ultilizada para A declaração de tipo de documento no HTML, também conhecida como DOCTYPE , é a primeira linha de código necessária em qualquer documento, HTML ou XHTML. A declaração de DOCTYPE é uma instrução para o navegador da web que diz em qual versão do HTML a página é escrita.

  • HEAD0000 "..." : O elemento head representa uma coleção de metadados para o documento. Metadados são, como o próprio nome diz, dados sobre os dados. De forma resumida, você poderá informar o navegador alguns detalhes sobre o conteúdo da página.

  • BODY "..." : O body alonga a silhueta, ajudando a deixar o corpo mais feminino. Por isso, sugerimos usar o body de manga longa com saias longas, lápis ou midi. Também pode contar com outras combinações feitas com calças, shorts, botas, saltos e acessórios.

  • CLASS "..." : Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos).

  • ID "..." : Atribui um id ao elemento (um id deve ser único, ou seja atribuído a um único elemento).

  • STYLE "..." : Permite incluir elementos CSS (estilos) dentro da tag.

  • LANG "..." : O atributo global lang ajuda a definir o idioma de um elemento: a língua em que elementos não-editáveis são escritos, ou a língua em que elementos editáveis devem ser escritos pelo usuário.

  • TITLE "..." : Define o título do elemento.

  • ALT "..." : Define um texto alternativo e, por isso, é muito utilizado em imagens, auxilia nas práticas de SEO.

  • HILDEN : Oculta o elemento.

  • ALING "..." : Permite definir o padrão de alinhamento desse elemento, como por exemplo: right, center, left e justify.

  • WIDTH "..." : Define uma largura para o elemento.

  • HEIGTH "..." : A propriedade height do CSS determina a altura da área do conteúdo de um elemento.

  • MAIN "..." : Que será o conteúdo principal de sua página, imagem, formulários, tabelas, textos, etc.

  • FOOTER "..." : Que será o rodapé do seu site, que pode apresentar também a Logo, algumas informações essenciais como número de contato, link para um fale conosco, etc.
